# Quick Case Engine

Convert text block casings instantly into Sentence Case, Title Case, UPPERCASE, lowercase, CamelCase, kebab-case, or snake_case.

## Features
- **All Standard Conventions**: Support UPPERCASE, lowercase, Sentence case, Title Case, camelCase, kebab-case, and snake_case.
- **Smart Title Casing**: Option to ignore prepositions and minor words (`of`, `the`, `in`, `and`, etc.) for publishing standards.
- **Format Preservation**: Retains structural paragraph breaks and spacings.
- **Privacy Guaranteed**: Runs completely in the browser; data never leaves your system.
