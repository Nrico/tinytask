# Email Signature Generator

Build clean email signatures that render consistently across all email clients, including Outlook's rendering engine (MSWord), Gmail, and Apple Mail.

## Features

- **Outlook-Safe Code**: Strictly table-driven layouts, inline CSS parameters, and absolute dimensions.
- **Copy Rich Text Signature**: Clipboard copy via standard `ClipboardItem` API so users can paste the actual styled signature straight into their settings.
- **Custom Color Swatches**: Allows setting custom Hex codes for primary theme elements.
- **Multiple Layout Choices**: Horizontal and Vertical layout splits.
