# Data Payload Sanitizer

Extract raw data from clipboard, stripping nested styling structures (HTML/XML/RTF) generated by Microsoft Office applications (Word, Excel, Outlook).

## Features

- **Clipboard Capture**: Listens to paste events and splits mime types.
- **Bloat Percentage Gauges**: Calculates byte sizes of raw `text/html` structures versus pure `text/plain` formats.
- **Discarded Tag Inspector**: Shows the raw inline HTML headers and class styling blocks discarded.
- **Auto-Copy back**: Automatically copies the clean formatted text back to your clipboard instantly on paste.
