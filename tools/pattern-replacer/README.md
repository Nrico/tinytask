# Pattern Replacer

Find and replace text using custom text, regex patterns, or plain-English checkboxes (Tabs, Line Breaks, double spaces) with yellow highlights.

## Features

- **Text Search & Replace**: Standard Find & Replace with case sensitivity and whole word filters.
- **Regex Mode**: Support full regular expression pattern replacements with real-time check validations.
- **Plain-English Presets**: Easily replace Tabs (`\\t`), Line Breaks (`\\n`), and collapse spaces without writing regex symbols.
- **Match Highlighter**: Visual rendering panel highlighting all active search matches in yellow.
