# Visual Style Purger

Strip inline styling, custom line-heights, and background colors from rich text while preserving clean structural paragraph breaks.

## Features
- **Absolute Plain Text**: Strips all HTML tags, outputting clean plain text.
- **Clean Structure**: Preserves basic markup (bold, italic, list items, simple paragraphs) but strips all custom attributes, background colors, custom fonts, and line heights.
- **Markdown Conversion**: Converts clean structural markup directly into standard Markdown syntax.
- **Bloat Metrics**: View exact character size reduction statistics showing how much metadata was removed.
- **Privacy First**: Processing happens 100% locally in the browser; data never leaves your system.
