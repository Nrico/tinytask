# Plain-Text List Un-breaker

Snap lines to the left margin, remove erratic indentations, and normalize bullet characters into clean plain-text list elements.

## Features
- **Format Normalizer**: Replaces Microsoft Word's erratic indentations and list formats.
- **Unified Bullet Prefixes**: Convert any bullet symbol (such as unicode bullets, dashes, checkboxes) into uniform plain-text symbols: `-`, `*`, numbered sequence `1. 2. 3.`, or completely clean text.
- **Whitespace Collapsing**: Collapse double spaces, trim outer tabs, and clean up empty line arrays.
- **Privacy Guaranteed**: Runs completely in the browser with no server-side writes.
